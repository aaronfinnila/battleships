# Battleships

A Battleships game built with JavaFX and Maven.

## Requirements

- Java 21
- Maven

## Build

```bash
mvn clean compile
```

## Run

```bash
mvn javafx:run
```

## Gameplay

Local multiplayer Battleships for two players on the same device.

1. Hide your ships - Each player places their ships on their grid in secret
2. Take turns - Players alternate shooting at each other's fleet, not knowing ship locations
3. Sink to win - Destroy all enemy ships to win

The right-hand grid shows the status of your own board, while the main canvas shows your opponents.
